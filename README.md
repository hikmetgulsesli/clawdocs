# ClawDocs

OpenClaw agent documentation system. Automatically scans all agents' capabilities, skills, model info, and tools, and serves them via a web dashboard.

## Architecture

- **Frontend**: React + Vite + TypeScript (port 3504)
- **Backend**: Node.js + Express + TypeScript (port 4504)
- **Shared**: Common types and utilities

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Typecheck

```bash
npm run typecheck
```
